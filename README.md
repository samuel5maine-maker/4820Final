Code is split across three different notebooks to prevent any singular one 
from being overwhelming long (this was primarily for my workflow)

Requirements:
Seaborn 0.13.2
Matplotlib 3.10.9
sklearn: 1.8.0
pandas:  3.0.2
numpy:   2.4.4

Notebook CSE4820_FinalProj contains the data exploration for the gaming regression problem
Notebook CSE4820_FinalProjModel contains the actual models and training for the gaming regression problem
CSE4820_DiabetesProj contains both the data exploration and models for the Diabetes Classification problem

All three notebooks can be run just by opening the notebook in VsCode or Colab and hitting "Run All"

Note: Once you run the notebooks all figures will be automatically saved into working directory in a file title figures.
